🚀项目经历

深渊逃生 · ABYSS ESCAPE

React TypeScript Zustand Framer Motion
2024.10 — 2025.01

一款将经典寻路算法游戏化的交互式 Web 应用。玩家在 20×20 网格上自由绘制地形， 选择算法实时观察探索过程与最优路径动画；融合体力消耗系统、战争迷雾、关卡预设及 S/A/B/C/F 成就评定，使算法演示具备完整游戏体验闭环。

  • 独立实现 BFS、Dijkstra、A* 三种寻路算法，支持地形权重（泥沼 cost=3）； 手写泛型 MinHeap<T>（二叉堆）为 Dijkstra/A* 提供 O(log n) 优先级队列，零第三方依赖。
  • 设计 5 档速度 + 暂停/续播动画引擎（idle→running⇌paused→done 状态机）， 引入 generation 机制解决 setTimeout 竞态问题， 通过 progressRef 断点精确续播，400 节点同屏流畅运行。
  • 基于 Zustand 拆分网格/游戏双 Store，配合 React.memo + 自定义比较函数， 单次算法动画期间减少约 90% 无效重渲染。
  • 实现战争迷雾（曼哈顿距离视野计算）、 鼠标拖拽绘墙（lastCellRef 去重防抖）、 4 关卡预设及自由绘图模式。
